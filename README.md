# JOMODCode
Old Java code for reading Novalogic's Joint Operations game files, game bot program 

DemoViewer
	Program made to load/view assets of the game
	Can read the archive format of the game, load models and textures, terrain files
	No runnable in its current state, but the code can help to make your own tools.
	Look at n3di and resource for loaders for certain file formats.

JOBotz
	Program to have more dynamic NPC opponents in the game. See the readme.txt
	The addon was released first in 2006, that version is not included here.
	jb2_201801	This is the released version, somewhat playable
	jb2_201811	This is the last updated version, less playable
